C 언어 기초 학습 계획서

목표
- C 언어의 기본 문법을 이해한다.
- gcc를 사용해 터미널에서 직접 컴파일하고 실행할 수 있다.
- 포인터, 배열, 문자열, 구조체의 동작 원리를 익힌다.
- 작은 콘솔 프로그램을 직접 작성할 수 있다.

프로젝트 디렉터리 규칙
- C 소스 파일 위치: ./c_code
- C 실행 파일 위치: ./c_run
- 기본 컴파일 예시:
  gcc -Wall -Wextra -std=c11 -o c_run/hello c_code/hello.c
- 기본 실행 예시:
  ./c_run/hello

학습 루틴
- 새 개념을 배운다.
- c_code 디렉터리에 예제 파일을 만든다.
- gcc로 컴파일한다.
- c_run 디렉터리에 실행 파일을 만든다.
- 실행 결과를 확인한다.
- 에러가 나면 메시지를 읽고 수정한다.

권장 파일 이름 규칙
- 01_hello.c
- 02_variables.c
- 03_if.c
- 04_loop.c
- 05_function.c
- 06_array.c
- 07_string.c
- 08_pointer.c
- 09_struct.c
- 10_project.c

컴파일 예시
- gcc -Wall -Wextra -std=c11 -o c_run/01_hello c_code/01_hello.c
- gcc -Wall -Wextra -std=c11 -o c_run/02_variables c_code/02_variables.c
- gcc -Wall -Wextra -std=c11 -o c_run/03_if c_code/03_if.c

주의할 점
- 경고 메시지를 무시하지 않는다.
- 실행 파일은 c_run에 둔다.
- 소스 파일은 c_code에 둔다.
- 한 번에 많은 문법을 외우기보다 직접 컴파일하고 수정한다.
- 포인터는 천천히, 주소와 값을 구분하면서 학습한다.
